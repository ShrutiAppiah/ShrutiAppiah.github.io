My Work

Crum & Forster

Integration Engineer
At Crum & Forster, I got to work with IBM Websphere products to integrate large enterprise applications. I created reports on transactional data using Microsoft SSRS/SSIS, the Business Intelligence Dev Studio, and other ETL tools. I also redesigned the UI of an internal web application to improve user experience.

Evertz

Project Engineer
At Evertz, I built a system that automated all the processes of a non-linear workflow for an online media-streaming platform using the Brightcove API. I also configured dynamic system-based reports, managed and added new features for NBC Channel's Content Management System. I also got to demo the final product to our clients.

Infosys

UI/UX Developer
At Infosys, I built a data visualization dashboard that provided insights on the jobs and tasks in an IT Service Automation Platform. The platform was made in an effort to optimize IT Support through automating repetitive tasks. I used D3.js, HTML/CSS, and Javascript to build a fully customizable, interactive, and dynamic dashboard.
